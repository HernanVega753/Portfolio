Sobre mí

Como programador

Soy Técnico Universitario en programación egresado de la Universidad Tecnológica Nacional de San Rafael, Provincia de Mendoza, Argentina. Allí desarrollé muy sólidas bases en lógica de programación y metodologías ágiles de trabajo en equipo. Además de dominar diferentes lenguajes sino también a implementar tecnologías fundamentales para el desarrollo como control de versiones, principios de diseño de software, frameworks, entornos virtuales, contenedores y testing.
Estas sólidas bases han facilitado mi recorrido autodidacta tomando contacto con aún más tecnologías como Godot Engine para videojuegos, PHP, desarrollo de aplicaciones low code y proyectos personales variados.
en la programación encontré maravillosas herramientas para poder hacer volar mi imaginación y crear todo lo que pueda nacer de mi mente.

Proyectos

Sistema Registro de Estudiantes (Java, Spring)

Sistema basado en listar, crear, modficicar y eliminar registros (CRUD). Se usó Java con Spring y Spring Boot, con MySql como base de datos

Sistema Tienda de Libros (Java, Spring, Swing)

Sistema para tienda de libros que permite el listado, creación, modificación y eliminación de libros (CRUD). Se usó Java con Spring y Spring Boot, UI realizado con Swift y base de datos MySql

Juego realizado en Godot Engine utilizando el lenguaje del motor (GdScript). Juego de un pequeño perro que tiene como misión alejar a las personas de su territorio.

Recuperador de Credenciales para WiFi (Python)

Programa simple realizado en Python para recuperar las credenciales de WiFi de la computadora que lo utiliza

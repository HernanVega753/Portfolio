Sobre mí

Como programador

Soy Técnico Universitario en programación. Si bien tengo formación FULLSTACK, es el BACKEND el que realmente me apasiona. Soy egresado de la Universidad Tecnológica Nacional de San Rafael, Provincia de Mendoza, Argentina. Allí desarrollé muy sólidas bases en lógica de programación y metodologías ágiles de trabajo en equipo. Además de dominar diferentes lenguajes también aprendí a implementar tecnologías fundamentales para el desarrollo como control de versiones, principios de diseño de software, frameworks, entornos virtuales, contenedores y testing.
Estas sólidas bases han facilitado mi recorrido autodidacta tomando contacto con aún más tecnologías como Godot Engine para videojuegos, PHP, desarrollo de aplicaciones low code y proyectos personales variados.
en la programación encontré maravillosas herramientas para poder hacer volar mi imaginación y crear todo lo que pueda nacer de mi mente.

Proyectos

E-commerce Such Great Food

Proyecto realizado en prácticas. Fue realizado con HTML, CSS y JavaScript. El backend fue realizado con node.js y express. Para la seguridad se implementó jwt para restringir el acceso a ciertas secciones. Las contraseñas se encriptan con Bcrypjs y se guardan en base de datos MySql. Hace distinción de usuarios, para usuarios admin se realizó CRUD completo para productos. Se añadió API de MercadoPago para realizar los cobros. Se utilizó arquitectura MVC (Módelo Vista Controlador)

E-commerce Constructora ByteBusters

E-commerce realizado como integrador final para la Tecnicatura Universitaria en Programación de la UTN. El Frontend fue realizado con HTML, CSS y JavaScript. En el Backend se implementó node.js y express. Este proyecto cuenta con restricción de acceso mediante token (jwt), encriptado de contraseñas y CRUD completo de la base de datos. También se implementó MercadoPago como forma de pago. Se utilizó arquitectura MVC (Modelo Vista Controlador)

Sistema Registro de Estudiantes

Proyecto realizado en la Universidad Tecnológica Nacional. Sistema basado en listar, crear, modficicar y eliminar registros (CRUD). Se usó Java con Spring y Spring Boot, con MySql como base de datos. El proyecto se ejecuta en consola y permite agregar alumnos para ser guardados en base de datos (MySql) además de ver las entradas en base de datos, modificar elementos y eliminarlos.

Sistema Tienda de Libros (Swing)

Sistema de administración para tienda de libros que permite el listado de libros en stock, inputs para crear nuevas entradas de libros, posibilidad de modificar libros ya resgistrados y eliminación de libros (CRUD). Se usó Java con Spring y Spring Boot, UI realizado con Swift y base de datos MySqlEl sistema cuenta cuenta con control de errores y excepciones, además de interfáz grafica intuititava y simple de usar.

Juego Firuland

Juego realizado utilizando el motor gratuito Godot Engine haciendo uso del lenguaje del motor (GdScript). Fue realizado íntegramente por sprites, música y recursos gratuitos y elementos propios desarrollados por photoshop. Utilicé la arquitectura singleton para ordenar el sistema de nodos del motor de videojuegos Godot.

Recuperador de Credenciales (WiFi)

Programa simple realizado en Python para recuperar las credenciales de WiFi de la computadora que lo utiliza. La motivación de este este programa fue la posibilidad de recuperar contraseñas de WiFi que se encuentren guardadas en la memoria de la computadora pero que sean de dificil acceso. Si bien el programa es simple, lo incluyo porque solucionó un problema práctico que tuve.
